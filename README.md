# Tienda Computacion
Presentamos un proyecto de ecommerce basico para el curso de React en Coderhouse.
Esta aplicacion permite ver una lista de productos de computacion y realizar compras en linea, almacenando los productos y las ordenes de compra en firebase.

## Tecnologias utilizadas
- Vite
- React
- React-Router-Dom
- Firebase
- Toastify
- Bootstrap

## Funcionalidades
- Ver una lista de productos disponibles por stock
- Agregar productos a un carrito de compras
- Realizar una orden de compra con datos de usuario
